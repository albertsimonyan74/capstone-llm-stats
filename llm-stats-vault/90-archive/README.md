# 90-archive — Historical content

This directory preserves superseded or completed-one-shot project artifacts
that no longer have active consumers but retain provenance value.

## Subdirectories

- **_originals/** — Original sprint log files preserved before merging into
  sprint-history-aggregated.md (2026-05-02 cleanup execution).
- **audit_history/** — Superseded audit reports (e.g., day2_audit_report.md).
  Findings folded into comprehensive_audit_2026-05-01.md before archive.
- **intermediate_analyses/** — One-shot diagnostic outputs and planning docs
  (e.g., headline_numbers_500_diagnosis.md, perturbation_generator_plan.md).
  Includes scripts/ subdirectory for archived one-shot scripts.
- **legacy_flask_website/** — Pre-React Flask app (capstone-website/ supersedes).
- **proposal_provenance/** — Original RQ definitions and proposal-trace docs
  (Phase 1B reframed RQ structure; originals preserved for IEEE paper history).
- **phase_1c_superseded/** — B3-stratified self-consistency artifacts
  (existed prior to 2026-05-02 cleanup; canonical Phase 1C lives in
  experiments/results_v2/).
- **repo_root_data_mirror/** — (CREATED LATER, gated on D-5 Render path
  cleanup) Repo-root experiments/results_v2/ workaround mirror.

About This Research

This DS 299 capstone project evaluates the Bayesian statistical reasoning capabilities of four leading large language models using a benchmark derived from 7 graduate-level textbooks and 40 lecture materials. The benchmark covers 31 task types across numerical accuracy, method selection, assumption compliance, prompt robustness, and confidence calibration.

Research Questions

  • RQ1 Numerical & Statistical Accuracy — What is the numeric accuracy of each LLM on a standardized Bayesian task benchmark?
  • RQ2 Method Selection Accuracy — What proportion of tasks involve incorrect statistical method selection vs. the controlled baseline?
  • RQ3 Assumption Compliance — To what extent do LLMs identify and respect required statistical assumptions?
  • RQ4 Robustness to Prompt Variations — How stable are LLM outputs under paraphrasing, renaming, and adversarial perturbations?
  • RQ5 Confidence & Trust Calibration — Is model-reported confidence correlated with actual correctness in statistical tasks?

Methodology

1 Ground truth computation via closed-form Python functions for each conjugate model
2 Task specification generation (tasks.json) with numeric targets and rubric keys
3 Structured prompt generation for all 31 task types via prompt_builder.py
4 LLM querying via unified model_clients interface (Claude/Gemini/OpenAI/DeepSeek APIs)
5 Automated scoring: numeric tolerance + structure keyword checks + rubric key matching
6 Statistical analysis & visualization of results across model families and task types

Dataset

136 tasks across 31 task types, 4 tiers
126 numeric tasks + 10 conceptual tasks
7 Textbooks
◆ Bolstad — Introduction to Bayesian Statistics (2nd ed.)
◆ Bishop — Pattern Recognition and Machine Learning
◆ Ghosh, Delampady & Samanta — An Introduction to Bayesian Analysis
◆ Hoff — A First Course in Bayesian Statistical Methods
◆ Carlin & Louis — Bayesian Methods for Data Analysis (3rd ed.)
◆ Goldstein & Wooff — Bayes Linear Statistics
◆ Lee — Bayesian Statistics: An Introduction (4th ed.)
40 lecture materials from graduate courses
4 models: Claude, Gemini, ChatGPT, DeepSeek
